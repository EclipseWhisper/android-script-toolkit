# Android Script Toolkit

🚀 一套面向 Android 高级用户的 Shell / Root 脚本工具集合

---

## ✨ 功能

- 🔍 Root 环境检测
- ⚙️ CPU 频率调节
- 📜 系统日志导出（logcat / dmesg / pstore）
- 📦 模块辅助脚本（Tricky Store / 自定义配置）
- 🧰 通用工具脚本
